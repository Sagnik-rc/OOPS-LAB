## List of Programs :

### **1. Aim of the program**
Accept 10 numbers from command line and check how many of them are even and how many are odd.

**Input:**  
Enter 10 numbers from command line

**Output:**  
Display number of even numbers  
Display number of odd numbers

---

### **2. Aim of the program**
Program to find number of objects created out of a class using `static` modifier.

**Input:**  
Number of objects created in the program

**Output:**  
Display the number of objects created (e.g. no of objects = 3)

---

### **3. Aim of the program**
Write a class file `Box` with three data members (length, width, height) and a method `volume()`.  
Create an application class `Demo` where an object of `Box` class is created using user input and volume is printed.

**Input:**  
Enter length, width and height

**Output:**  
Display volume of the box

---

### **4. Aim of the program**
Write a program in Java to create a class `Rectangle` having data members length and breadth and three methods `read()`, `calculate()` and `display()` to read values, calculate area and perimeter, and display the result.

**Input:**  
Enter length and breadth

**Output:**  
Display area of rectangle  
Display perimeter of rectangle

---

### **5. Aim of the program**
Write a program in Java to input and display the details of `n` number of students having roll number, name and CGPA as data members. Also display the name of the student having the lowest CGPA.

**Input:**  
Enter roll number, name and CGPA of `n` students

**Output:**  
Display details of all students  
Display name of student with lowest CGPA

---

### **6. Aim of the program**
Write a program to overload `subtract()` method with various parameters in a class in Java and use it through a driver class.

**Input:**  
Mention different subtract methods with different parameters

**Output:**  
Display subtraction result based on the method called

---

### **7. Aim of the program**
Write a program in Java using constructor overloading to calculate the area of a rectangle.  
Use default constructor to initialize values to zero and parameterized constructor to initialize values using user input.

**Input:**  
Enter length and breadth

**Output:**  
Display area of rectangle
